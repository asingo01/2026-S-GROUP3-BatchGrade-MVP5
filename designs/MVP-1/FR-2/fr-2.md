# FR2 File Submission Button & Display

## Description 
This button cannot be functionally used until a file is uploaded using FR1. An error message will appear if the user tries to submit without first uploading a file. Upon proper use, submission information will be created/compiled and stored in FR7 and FR13. This information includes a unique ID for the submission, the submitter's ID and name, and the file name.
